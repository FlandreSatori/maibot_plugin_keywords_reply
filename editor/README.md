# 词库外部编辑器

在浏览器中管理 `keywords.json`：总览、筛选、搜索，以及批量增删改。

## 启动

在仓库根目录执行（将 `--data-dir` 换成你的 MaiBot 插件数据目录）：

```bash
python editor/server.py --data-dir ".../MaiBot/data/plugins/maibot_plugin.keywords_reply"
```

或者直接启动``editor.bat``

默认监听 `http://127.0.0.1:8765`。

## 功能

- **总览**：关键词 / 检测词分页签切换，统计数量
- **筛选**：启用状态、需@、正则、群号、全文搜索（找关键词）；编辑弹窗内可按回复类型筛子词条
- **批量操作**：启用、禁用、开启/关闭需@、**批量群策略**、删除
- **单条编辑**：触发词、群黑白名单、结构化回复行（文本 / 音乐 / 图片 / 语音 / 表情 + 概率%）
- **导入 / 导出**：JSON 文件

## 大词库性能

7000+ 词条时编辑器采用：

- **列表分页**（默认每页 50，可调 30/100/200）
- **搜索防抖**（约 280ms，避免每键重绘）
- **回复列摘要**（显示条数 + 首条预览，不再展开全部回复）
- **编辑弹窗内回复分页**（每页 40 条）

首次加载超大 JSON 仍可能需要数秒；加载完成后翻页、筛选会流畅很多。建议用搜索缩小范围后再编辑。

## 回复编辑（结构化）

编辑词条时，**每条带边框的回复 = 一个 `entry`**，内部是紧凑行（类型 | 内容 | 概率% | 操作）。

- 最后一行末尾 **+** 弹出菜单，可多次添加文本/图片/语音/表情/音乐（顺序即发送顺序）
- **×** 移除该行素材；勾选 entry 后点 **删除选中** 批量删除（按钮在「添加回复」旁）
- 长按行左侧 **⠿** 可拖动调整同一条 entry 内的发送顺序
- 弹窗内 **「全部回复 / 含文本 / 含图片…」** 只筛选当前关键词下的子回复

主列表用左侧搜索找关键词即可。

## 批量群策略

勾选多个词条 → **批量群策略**：

- **替换为以下策略**：统一设置白/黑名单与群号
- **在现有群号上追加**：保留原群号并追加新群号
- **设为全局启用**：黑名单模式且群列表为空
- **设为全局禁用**：关闭该词条

## 与 MaiBot 同步

编辑器直接写入 `keywords.json`。保存后请在群聊执行 **`/重载词库`**，或重启 MaiBot，插件才会读取最新数据。
